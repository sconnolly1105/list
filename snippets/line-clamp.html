Example of truncatin text to any number of lines

    You can truncate the below text to any the beow text to any number of lines buy using the -webkit-line-clamp css as shown below

    <p class="clamped">
      In this example the -webkit-line-clamp property is set to 3, which means the text is clamped after three lines.
      An ellipsis will be shown at the point where the text is clamped.
    </p>

  <style>
  .clamped {
    margin-top: 200px;
    width: 300px;
    display: -webkit-box;
    -webkit-box-orient: vertical;
    -webkit-line-clamp: 3;
    overflow: hidden;
  }
  </style>


  Note that -webkit-line-clamp: none; can also be used
