Below you’ll see an example where we have an image element being observed. When 50% of the image is visible in the viewport upon scrolling, the fancy class gets added causing the image to spin.
You will need to scroll down until 50% of the image is visible
The 50% is defined in the below threshold. As this is an array, various items can be added. So something like { threshold: [.25, .5, .75] } would cause the observer to fire whn 25%, 50% and 75% of the image was exposed. This means that discrete animations can be carried out depending on the amount of the image that is exposed in the viewport.
